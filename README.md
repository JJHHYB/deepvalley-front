## 🌲 Git Branch - Frontend Repository

`main` : 서비스 운영 브랜치 </br> `develop` : FE 개발 환경 브랜치 </br>
`feat/기능명` : 기능 개발 브랜치 ex) feat/Login </br> `fix/버그명` : 버그 수정
브랜치 ex) fix/Login </br> `refactor/모듈명` : 리팩토링 브랜치 ex)
refactor/Login </br>
